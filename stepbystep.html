Git Ready, Set, Code! Git and Github Setup Step-by-Step Guide

Fast start for Windows users: install, configure, connect to GitHub, and run essential commands (init, clone, add, commit, push, pull, branch).

Download Git on Windows

  • To download Git, you should have a later version of Windows on your computer device.
  • Make sure you have a Github account.

1) Install Git for Windows

To start, go to the official Git website and download the installer suitable for your Windows system.

You can download it here: https://git-scm.com/download/win

2) Click 64-bit Git under Standalone Installer

On the download page, scroll down and click on the 64-bit Git for Windows Setup link to download the 64-bit installer.


This will download the latest version of Git for Windows. Always choose the 64-bit setup for modern systems.

3) Install the Downloaded File

After the download completes, locate the setup file in your browser‚Äôs Downloads section or through the File Explorer. Run the installer and follow the default recommended settings to finish installation.

4) Verify if Git is Installed

Once installation is complete, create a folder anywhere on your computer. Then, right-click inside the folder to check if Git is properly installed.

Note: For Windows 10, you‚Äôll immediately see the installed Git options upon right-click. For Windows 11, click "Show more options".


If you can see both ‚ÄúOpen Git Bash here‚Äù and ‚ÄúOpen Git GUI here‚Äù options, Git is successfully installed!

5) Complete the Installation

When Git Setup installation appears, keep clicking Next until the installation begins. After that, click Install and wait for the process to finish. When the installation is completed, click Finish. You can now proceed to set up your GitHub account and start managing your repositories.

Setting Up Your GitHub Account

  1. Go to a Github website GitHub.com and create a new GitHub account or log in using your active Google account.
  2. After logging in, click the green New button on the left side of the dashboard to create a new repository.
  3. Enter a repository name and click Create Repository located at the bottom right of the page.

üí° Tip: Keep repository names short, simple, uppercase and lowercase. Avoid using spaces; use hyphens (-) or underscores (_) instead.
